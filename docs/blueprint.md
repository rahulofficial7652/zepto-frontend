# **App Name**: Zepto Clone

## Core Features:

- Landing Page with Location & Search: A landing page that will welcome the user, automatically figure out location and propose corresponding product options.
- Dynamic Product Listing: Display products dynamically based on category.
- Product Detail Page: Detailed product view with image carousel, description, and quantity control.
- Shopping Cart Management: Manage items in the cart, update quantities, and calculate totals.
- Simple Checkout Process: Collect address and simulate payment processing with loading animations.
- Order History: Display past orders with status tracking (delivered, processing, cancelled).
- User Authentication: Implement OTP-based login/signup UI.

## Style Guidelines:

- Primary color: Vibrant orange (#FF8C00) to reflect energy and immediacy of delivery.
- Background color: Light gray (#F7F8FA) for light mode and dark gray (#1A1A1A) for dark mode.
- Accent color: A slightly desaturated orange (#FFB347) for interactive elements and highlights.
- Body and headline font: 'Inter' for a modern and neutral feel.
- Use Lucide icons for a consistent and clean UI experience.
- Employ cards with rounded-2xl corners and soft shadows for depth. Maintain clean spacing (p-4 to p-6) throughout.
- Incorporate Framer Motion transitions (fade, slide, and scale) between pages for a smooth user experience.
